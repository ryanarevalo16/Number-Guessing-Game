# Number Guessing Game (Java)

## Overview
This is a simple console-based number guessing game built in Java. The player selects a difficulty level and tries to guess a randomly generated number within a limited number of attempts.

The game provides feedback after each guess, indicating whether the guess is too high or too low, and tracks remaining attempts until the player either wins or runs out of guesses.

---

## Features
- Difficulty selection:
  - Easy (10 guesses, 1–50)
  - Medium (7 guesses, 1–100)
  - Hard (5 guesses, 1–200)
- Limited number of attempts based on difficulty
- Input validation to prevent crashes from invalid input
- Win/loss detection
- Replay option after each game
